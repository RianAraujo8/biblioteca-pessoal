# Matriz de Rastreabilidade de Requisitos

| ID | Requisito Funcional | Descrição | Casos de Teste | 
|----|---------------------|----------|----------------|
| R1 | Criar livro | Permitir cadastrar um livro | CT01, CT02 | 
| R2 | Listar livros | Retornar todos os livros | CT03 | 
| R3 | Atualizar livro | Editar dados do livro | CT04, CT05 | 
| R4 | Deletar livro | Deletar livro | CT06, CT07 | 
| R5 | Cadastrar usuário | Criar novo usuário | CT08 | 
| R6 | Listar usuários | Retornar todos os usuários | CT09 | 
| R7 | Atualizar usuário | Editar dados do usuário | CT10, CT11 | 
| R8 | Deletar usuário | Deletar usuário | CT012, CT13 | 

## Casos de Teste

- CT01: Deve cadastrar livro com dados válidos
- CT02: Deve recusar cadastrar livro com dados inválidos
- CT03: Deve listar livros cadastrados
- CT04: Deve atualizar um livro existente
- CT05: Deve recusar atualizar um livro inexistente
- CT06: Deve deletar um livro existente
- CT07: Deve recusar deletar um livro inexistente
- CT08: Deve cadastrar usuário válido
- CT09: Deve recusar cadastrar usuário com dados inválidos
- CT10: Deve atualizar um usuário existente
- CT11: Deve recusar atualizar um usuário inexistente
- CT12: Deve deletar um usuário existente
- CT13: Deve recusar deletar um usuário inexistente
